Star Wars Quiz App

This app was a beginning challenge from my Thinkful course. The challenge was to make a small dynamic quiz app by using JavaScript to store all the questions and answers in a local object that can be used to display the correct information as well as communicate when the user has selected the correct answer. I chose one of my favorite Sci-Fi universes to base all of the questions on! This simple web app uses a mix of HTML, CSS, Javascript and JQuery.
